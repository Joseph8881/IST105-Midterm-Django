# Django Calculator App (AWS Deployment)

## Features
- Simple calculator (add, subtract, multiply, divide)
- Django-based web app
- Deployed on AWS EC2
- Uses Gunicorn + Load Balancer

## URL
http://<your-load-balancer-dns>/calculate/

## Tech Stack
- Django
- Gunicorn
- AWS EC2
- Application Load Balancer
- Auto Scaling Group

## How to Run Locally

```bash
python manage.py runserver
